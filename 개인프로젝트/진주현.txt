<예약 관리 프로그램>
1. 데이터베이스
 => 회원 정보 : 주민번호(PK), 이름, 전화번호, 생년월일, 주소, 예약번호(FK)
 => 예약 정보 : 예약번호(PK), 예약날짜, 예약내용
​
2. 구현 기능
1) 회원 및 예약 정보 저장
 => 회원 : 주민번호, 이름, 전화번호, 생년월일을 입력받아 저장
 => 예약 : 예약번호, 예약날짜, 주민번호를 입력받아 저장
 => 입력값에 대한 유효성 검사
​
2) 회원 및 예약 정보 변경
 => 주민번호를 입력받아 회원 및 예약 정보 검색, 검색 정보가 없는 경우 메시지 출력
 => 주민번호, 이름, 전화번호, 생년월일을 입력받아 저장
 => 예약번호, 예약날짜를 입력받아 저장
 => 입력값에 대한 유효성 검사
​
3) 회원 및 예약 정보 삭제
 => 주민번호를 입력받아 회원 및 예약 정보 삭제, 정보가 없는 경우 메시지 출력
 => 입력값에 대한 유효성 검사
​
4) 이름 검색
 => 이름을 입력하여 회원정보 및 예약 정보 검색, 정보가 없는 경우 메시지 출력
 => 입력값에 대한 유효성 검사

<상담 관리 프로그램>
1. 데이터베이스
 => 상담 정보 : 상담코드(PK),담당자,이름,전화번호,주소,상담날짜,상담진행상태
 => 상담진행상태 컬럼에는 문자값 0 : 상담예약상태, 1 : 상담완료

2. 구현 기능
1) 상담 정보 저장
 => 상담코드, 이름, 전화번호, 생년월일,주소,상담날짜,상담내용을 입력받아 저장
 => 입력값에 대한 유효성 검사
2) 상담 정보 변경
 => 상담코드를 입력받아 상담 정보 변경, 정보가 없는 경우 메시지 출력
3) 상담 정보 삭제
 => 상담코드를 입력받아 상담 정보 삭제, 정보가 없는 경우 메시지 출력
4) 상담 정보 검색
 => 상담코드,이름을 입력받아 상담 정보 검색, 정보가 없는 경우 메시지 출력
 => 입력값에 대한 유효성 검사
